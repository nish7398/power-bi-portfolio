# power-bi-portfolio
## Project 1: HR Data Cleaning

**Tools:** Power BI, Power Query
**Dataset:** 83 rows of messy HR employee data

**Cleaning Tasks:**
- Removed null values in critical columns (Employee Name, Hire Date)
- Replaced null salaries with 0
- Standardized capitalizations (IT, HR, ACTIVE/INACTIVE)
- Removed duplicate entries
- Final: 71 clean rows ready for analysis

**Skills Demonstrated:**
- Data quality assessment
- Power Query transformations
- Null value handling
- Duplicate removal
- Data standardization

**Files:**
- `assignment.pbix` - Power BI file with all transformations
- `Assignment_messy_Hr_data.txt` - Original messy data
- `assignment_final.xlsx` - Cleaned output
